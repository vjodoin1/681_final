# Math 681 Final Project

This repository contains materials from the final project for **Math 681: Advanced Mathematical Biology I** at the University of Tennessee, Knoxville (Fall 2025).

## Repository Structure

```
├── matlab/          # MATLAB code and scripts for the project
├── docs/            # Final project PDF and documentation
└── README.md        # This file
```

## Contents

- **MATLAB Code**: All MATLAB scripts and code are located in the `matlab/` directory
- **Documentation**: The final project PDF and related documentation are in the `docs/` directory

## Usage

1. Navigate to the `matlab/` directory to access the MATLAB code
2. Navigate to the `docs/` directory to access the final project PDF

## Course Information

- **Course**: Math 681 - Advanced Mathematical Biology I
- **Institution**: University of Tennessee, Knoxville
- **Semester**: Fall 2025
